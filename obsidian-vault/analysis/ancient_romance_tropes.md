## 子标签组合规律

**高转化组合**：
- `HE + 1v1 + 古风世情` → 3本（安全感 + 专一 + 格调）
- `疯批/病娇 + 替嫁 + 权谋` → 2本（极致张力）
- `重生 + 女主搞事业/经营` → 2本（现代爽文逻辑套古装皮）

## 选题建议

1. **稳妥方向**：重生女主 + 朝堂权谋 + 1v1 HE + 轻经营线（参考《大渝筹算师》《逢灯纪》）
2. **爆款方向**：疯批男主 + 替嫁开局 + 女主看似弱实则强 + 权谋暗线（参考《笨蛋美人...》《指挥使...》）
3. **差异化**：女主视角纯经营/建国 + 无CP或晚CP + 群像戏（参考《穿乱世...建国了》稀缺性）
