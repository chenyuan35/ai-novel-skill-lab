## 组合公式

**古言爆款公式**：
```
疯批/病娇男主 + 替嫁/冲喜开局 + 女主看似弱实则强/重生 + 朝堂权谋线 + 1v1 HE
```

**现言爆款公式**：
```
高冷总裁/权势男主 + 契约/误会开局 + 女主事业线独立 + 先婚后爱/久别重逢 + 1v1 HE
```

**玄幻爆款公式**：
```
暴戾战神/魔尊男主 + 女主被误解/废柴逆袭 + 师徒/契约羁绊 + 升级流爽点 + 单女主
```

## 避雷指南（读者吐槽高频）

| 雷点 | 表现 | 规避建议 |
|------|------|----------|
| 男主前期渣/虐女主 | 追妻火葬场过长、虐度过高 | 轻虐重追、追妻不超5万字、男主自始至终护短 |
| 多女主/暧昧不清 | 红颜知己多、不拒绝他人好意 | 单女主铁律、男主对除女主外异性零容忍 |
| 女主圣母/软弱 | 事事依靠男主、不杀敌人留后患 | 女主有独立能力/金手指、杀伐果断 |
| 误会拖剧情 | 可沟通解决却硬拖几十章 | 误会3章内解开、靠信息差非性格缺陷 |
