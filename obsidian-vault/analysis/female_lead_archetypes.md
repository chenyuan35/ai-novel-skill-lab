## 核心爽点设计公式

**古言女主爽点**：
```
重生/穿越（信息差）+ 扮猪吃虎（反转爽）+ 权谋/经营（成就感）+ 男主强强联手（安全感） = 完整爽文闭环
```

**现言女主爽点**：
```
职场/事业线独立（现代女性共鸣）+ 情感线不拖沓（成熟恋爱观）+ 家庭/原生家庭破局（共情深） = 现实投射爽感
```

**玄幻/仙侠女主爽点**：
```
废柴逆袭/觉醒神体（升级爽）+ 师徒/契约羁绊（情感锚点）+ 打脸打脸再打脸（社交爽）+ 守护想守护的人（宏大意义） = 升级流标准公式
```

## 避雷指南（女主篇）

| 雷点 | 表现 | 规避建议 |
|------|------|----------|
| 圣母/软弱/只会哭 | 遇事靠男主、不敢杀敌、原谅渣男渣女 | 女主有底线、有手段、杀伐果断、不靠男主救场 |
| 无脑作/制造误会 | 明明能沟通却硬猜疑、为剧情强行降智 | 冲突源于信息差/立场差而非性格缺陷、沟通及时 |
| 事业线假/挂名 | 说搞事业实则靠男主给资源、不展示专业细节 | 真实展示专业过程、女主核心决策者、男主辅助而非主导 |
| 重生后不利用优势 | 知道剧情却不避坑、不囤资源、不提前布局 | 重生必须体现"信息差变现"、每个关键节点都要比上一世做得更好 |
| 穿越后不适应/只会卖惨 | 现代思维不落地、只会抱男主大腿 | 现代知识转化为古代可用的生产力/管理学/医术/法律 |

## 爆款女主"三要素"

1. **有爽点源头**：重生/系统/空间/马甲/特殊技能/信息差 —— 解释"凭什么她能赢"
2. **有成长弧光**：从弱到强、从被动到主动、从自保到庇护他人 —— 读者陪伴成长的满足感
3. **有独立人格**：感情线不跪舔、事业线不挂名、面对男主有底线有原则 —— 现代女性代入感
