# Novel Creation Workflow

## One Session Loop

1. Confirm the current novel and chapter target.
2. Read production status, director status, ledgers, and style context.
3. Update chapter brief, task sheet, scene cards, must-avoid rules, and target word count.
4. Generate through AI Novel.
5. Read the chapter in preview/sample form.
6. Ask the user to choose: pass, revise, or redirect.
7. If passed, mark the chapter complete and capture facts, hooks, style notes, and feedback.

## Recovery Loop

1. Check backend health.
2. Check provider/model route.
3. Check structured output and latest task state.
4. Check database state.
5. Retry only after the failing layer is named.
